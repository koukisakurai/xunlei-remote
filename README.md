# xunlei-remote
从 Chrome 工具栏快速打开迅雷远程设备页面，查看和管理远程下载任务。无需额外权限，不收集用户数据。

迅雷-远程下载扩展隐私政策

生效日期：2026年7月13日

一、扩展用途

“迅雷-远程下载”扩展的唯一用途，是在 Chrome 工具栏弹窗中提供迅雷远程设备网页的快捷入口。

二、开发者收集的数据

本扩展的开发者不会收集、读取、保存、分析或上传用户的个人信息、浏览记录、下载内容、账户信息、身份验证信息、Cookie、远程设备信息或下载任务信息。

本扩展不包含统计分析、广告追踪或用户行为追踪功能。

三、第三方服务

本扩展会在弹窗中加载以下第三方网页：

https://pan.xunlei.com/yc

当用户登录或使用迅雷远程设备服务时，账户信息、身份验证信息、Cookie、远程设备信息以及下载任务信息，可能由用户的浏览器直接发送给迅雷服务。

这些数据由迅雷网站直接处理。本扩展开发者无法访问、保存或分享这些数据。迅雷服务的数据处理行为受迅雷自己的服务条款和隐私政策约束。

四、浏览器权限

本扩展不申请浏览记录、标签页、Cookie、存储、下载管理、网页内容或主机访问权限。

本扩展不会读取用户正在浏览的网页，也不会在后台监控用户活动。

五、数据共享与出售

本扩展开发者不会向任何个人、企业、广告平台或数据经纪商出售、出租或分享用户数据。

本扩展不使用用户数据进行个性化广告、信用评估或其他与扩展单一用途无关的活动。

六、数据保留

由于本扩展开发者不收集或存储用户数据，因此不存在由开发者执行的数据保留或删除流程。

用户可以通过卸载本扩展停止使用该服务。迅雷账户和远程设备相关数据应通过迅雷提供的账户管理功能进行管理。

七、信息安全

本扩展加载的迅雷远程设备页面使用 HTTPS 加密连接。本扩展不会将用户数据发送到开发者控制的服务器。

八、隐私政策变更

如果本扩展的功能或数据处理方式发生变化，本隐私政策将同步更新。更新后的政策会发布在当前页面。
