## 扩展用途

“迅雷远程下载”扩展的唯一用途，是在 Chrome 工具栏弹窗中提供迅雷远程设备网页的快捷入口，使用户无需离开当前浏览页面即可访问迅雷远程下载界面。

## 功能特点

- 从 Chrome 工具栏快速打开迅雷远程设备页面
- 查看远程设备、下载中任务和已完成任务
- 使用迅雷网页提供的远程下载功能
- 基于 Manifest V3 构建，适配新版 Chrome
- 不包含广告、统计分析或用户行为追踪功能
- 不申请浏览记录、标签页、Cookie、存储或主机访问权限

## 使用方法

1. 安装扩展并将其固定到 Chrome 工具栏。
2. 点击“迅雷远程下载”扩展图标。
3. 根据迅雷页面提示登录账户并选择远程设备。
4. 在弹窗中查看或管理远程下载任务。

远程下载功能是否可用，取决于迅雷账户、网络环境以及绑定设备是否支持迅雷下载。
